# Underscore in Python

## Overview

This activity demonstrates five common uses of the underscore (`_`) in Python.
The examples are written in a single script with short comments to make each use clear.

## File

- `underscore.py`: contains all five underscore examples

## Concepts Demonstrated

1. Last expression in Python
2. Ignoring values
3. As a loop variable
4. Formatting large numbers
5. Placeholder for temporary or unimportant variables

## How to Run

Run the script with:

```bash
python underscore.py
```

## Example Output

```text

2. Ignored Middle: First=1, Last=5
3. Loop 5 times: ★
★
★
★
★

4. Large Number: 1000000000
5. Temporary Placeholder (fruit): apple
5. Temporary Placeholder (fruit): banana
```

## Notes

This script is a simple learning example.
Some uses of `_` are based on Python convention, while others are part of normal Python syntax.
